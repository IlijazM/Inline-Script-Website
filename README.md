# Inline-Script-Boilerplate

## Install

``` bash
npx degit IlijazM/Inline-Script-Boilerplate
```

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at 0.0.0.0:8080
$ npm run dev

# generate files and update all templates
$ npm run generate

# build for production
$ npm run build
```
