Inline Script - Button Syntax

Button syntax

Basics

There is a special syntax for the button element. If you use the inline script syntax on a button, the inline script will get executed whenever the client clicks on the button. To set the text on the button you can use the attribute value.

It looks like this:

<button value="CLICK ME">{{
    console.log('Hello, world!');
}}</button>

Practical example

Let's say we have the variable counter. Every time a button gets pressed the variable counter should get increased by one. The element with the id display should be updated automatically.

To do this we first need to define a div element with the id display. Using the inline script syntax we can tell the div that it should display the value of the variable counter.

Next, we want our button to first increment the variable counter and then to re-render the element.

Also make sure you define the variable counter in a script tag else this will throw an exception.

At the end we have some code that looks like this: 

<body>
    <div id="display">{{
        counter
    }}</div>

    <button value="Increment">{{
        counter++;
        document.getElementById('display').render();
    }}</button>

    <script>
        let counter = 0;
    </script>
</body>
