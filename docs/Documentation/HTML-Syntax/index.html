HTML Syntax

Basics

Inline-script.js provides a now syntax to your inline script code. It basically allows you to return HTML elements with ease. You can use this syntax by putting your html element inside parentheses. 

<div>{{
    (<h1>Hello, world</h1>)
}}</div>

Store html in variables

You can also set this to a regular variable like following: 

<div>{{
    let element = (<h1></h1>);
    element.innerHTML = 'Hello, world';
    element
}}</div>

This will return a new element with the content 'Hello, world'.

This also works with arrays. If you fill an array with HTMLElements, each one will get appended to the parent element.

Store html in arrays

<div>{{
    let elementA = (<h1>Hello</h1>);
    let elementB = (<h2>world</h2>);

    [elementA, elementB];
}}</div>

Nesting the inline script syntax

One of the most practical ways to use this syntax is to nest it in itself. Like with regular elements, an element that got defined by the new syntax can also use the inline script syntax like following: 

<div>{{
    (<h1>{ Math.random(); }</h1>)
}}</div>

Why is this useful? Maybe you want to generate different kinds of elements based on some input. In this example 50% of the time a red text will appear and the other 50% of the time a blue text will appear. This can be implemented as follows: 

<div>{{
    Math.random() > 0.5 ? (<p style="color:red;">red</p>) : (<p style="color:blue;">blue</p>)
}}</div>

It's also possible to use the inline syntax on a child of the html element.

Using local variables

If you define a variable inside the inline script syntax, you can access the variable inside this scope. For example you want to display a text based on some input that returns a random word. You can implement this as follows: 

<div>{{
    const text = ['lorem', 'ipsum', 'dolor', 'sit'][Math.floor(Math.random() * 4)];

    (<h1>{ text }</h1>)
}}</div>

You can nest these element indefinitely so this is a valid syntax as well: 

<div>{{
    const text1 = ['lorem', 'ipsum', 'dolor', 'sit'][Math.floor(Math.random() * 4)];

    (<div>{
        const text2 = ['lorem', 'ipsum', 'dolor', 'sit'][Math.floor(Math.random() * 4)];

        (<div>{
            text1 + ' ' + text2
        }</div>)
    }</div>)
}}</div>

Creating an array dynamically

A very good use case for this syntax is to dynamically generate a navigation bar based on a variable. In this example, we're going to define a variable called navigation and generate a navigation bar from it.

To do this we need to define our header. Inside the inline script syntax we're going to map the array and return a new element. At the end it should look something like this: 

<body>
    <header>{{
        navigation.map(item => {
            return (<a>{ item }</a>)
        })
    }}</header>

    <script>
        const navigation = ['Home', 'About', 'Pricing', 'Contact'];
    </script>
</body>
