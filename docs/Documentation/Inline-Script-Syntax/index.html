The inline script syntax

Basics

The inline script syntax is the core of the front-end framework. It feels similarly like the syntax in react-js.

Let's say you want to randomize the content of a div. The typically way to approach this is by adding an empty div that will later be filled with content and whenever a button got pressed, a function will get called. 

<body>
    <div id="random"></div>

    <button onclick="pressed()">CLICK ME</button>

    <script>
        const list = ['üßÅ', 'üç´', 'üç¨', 'üç≠', 'üç°'];

        function setContent() {
            document.querySelector('#random').innerHTML = list[Math.floor(Math.random() * 5)];
        }

        function pressed() {
            setContent();
        }

        setContent();
    </script>
</body>

As you can see, the actual code that describes the functionality of the div can sometimes be very far away. Inline-script.js has a solution for this. If you start the content of the div with a { (or {{ for syntax highlighting) you use the inline script syntax. This script will automatically get executed and the resulting value will be the content of the div 

<body>
    <div>{{
        ['üßÅ', 'üç´', 'üç¨', 'üç≠', 'üç°'][Math.floor(Math.random() * 5)];
    }}</div>
</body>

Using this syntax your html element are visually connected with the appropriate code which improves the structure of your html.

Further use cases

This could be very useful for displaying constant values like the company name:

<body>
    <div>{{companyName}}</div>

    <script>
        const companyName = 'My company';
    </script>
</body>

Restrictions

As with all good things this one also has some limits. You may not use the template literals syntax ( ` ) in your code.

It could also be the case that the page takes too long to render and hence your page reveals the initial inline script code. To bypass that you can set the display of the body to none by default and after you executed the function inlineScript you reveal the body. 

inlineScript();
document.body.style.display = 'block';
