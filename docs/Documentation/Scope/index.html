Scope

Basics

The variable scope points at the outer element in another file. You can use the variable to safely query select element that only appear in the current file. You can access this variable using the inline script syntax. 

<div>
    <h1></h1>
    <div>{{
        scope.querySelect('h1').innerHTML = 'Hello, world'
    }}</div>
</div>
