Inline Script - Scoped CSS

Scope CSS

Basics

Scoped css is an important feature if you'd like to put your elements into components. Luckily, Inline-script.js will automatically scope your styles. Scoped css is not a thing in regular html anymore because of the lack of browser support, however, if you would to apply the scope using javascript this now becomes a possible feature that you can use in html.

The scoped css in Inline-script.js works the same as in scoped-css-js so to see further documentation you can follow the link.

All you have to know is that you can put the attribute scoped to your style-tag and Inline-script.js will automatically scope the css. 

<body>
    <div>
        <h1>This is big</h1>
        <style scoped>
            h1 {
                font-weight: 800;
                font-size: 6em;
            }
        </style>
    </div>

    <h1>This isn't</h1>
</body>

The selector '#this'

Let's say you want to select the outer element (in this example the header). To accomplish that you can use the selector #this: 

<header>
    <p>Hello, world!</p>
    <style scoped>
        #this {
            background: black;
            color: white;
        }
    </style>
</header>

The attribute scope

You alternatively can use the attribute scope to set the scope manually. For example you want to select every a-tag thats inside a li-tag. 

<style scope="li a">
    #this {
        color: black;
    }
</style>
