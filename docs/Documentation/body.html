Setup

What makes {productName} so special is that it is relatively easy to embed:

Start by adding the inline-script.js in your head by either downloading it or linking to the github url.

<script src="https://ilijazm.github.io/Inline-Script/inline-script.js"></script>

then call the inlineScript function like this:

<script>inlineScript();</script>

Now your good to go to start your first { productName } project today!
