Updating an element

Basics

Let's say you want to update your element whenever e.g. a button got pressed. This is luckily very simple. You can select your element and then call the function render() to re-render it.

In this example we have a div that randomly outputs one emoji of a list. We can add a onclick attribute to it that calls the function update. Now all that's left is a way to select the element and to call the render function. We can select the element by the id random and voila. 

<body>
    <div id="random" onclick="update()">{{
        ['🧁', '🍫', '🍬', '🍭', '🍡'][Math.floor(Math.random() * 5)];
    }}</div>

    <script>
        function update() {
            document.getElementById('random').render();
        }
    </script>
</body>
