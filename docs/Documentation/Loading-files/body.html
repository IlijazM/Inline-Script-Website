Loading files

Load function

Inline-script.js provides a function called load which loads another file. It takes three parameters:

  1. First the element which usually is this.

  2. The second parameter is the url (e.g. 'header.html').

  3. The last parameter are some optional arguments in form of an object.

You can access these parameters by simply using the inline script syntax.

index.html

<div>{{
    load(this, 'header.html', { title: 'My website' })
}}</div>

header.html

<div>
    <header>
        <div class="title">{{title}}</div>
        <nav> ... </nav>
    </header>
</div>

Load attribute

There is also another way to load a file and to use the innerHTML of an element as an argument. You can use the attribute load to load another file. Additional arguments can be set by using attributes as well. The innerHTML of the original element will then be stored inside the variable html

index.html

<div load="big.html" color="red">
    Hello, world!
</div>

big.html

<div style="font-size: 6em; color: {{color}}">{{
    html
}}</div>

Proper use

Note that you can only use one element in a new file so this is not a valid syntax:

<h1>Hello</h1>
<h2>world</h2>

but this is:

<div>
    <h1>Hello</h1>
    <h2>world</h2>
</div>
